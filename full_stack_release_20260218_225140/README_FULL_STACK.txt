NeuroDSL Full Stack Bundle

Includes:
- Web console (web/web_champion_v2)
- Model server EXE (model_server/NeuroDSL_Champion_v2.exe)
- Console EXE (console/dist/NeuroDSL_Console.exe)
- GUI EXE (gui/NeuroDSL_Power_Champion_GUI_Plus.exe, if available)

Quick start:
1) Run start_web_stack_all.bat
2) Open http://127.0.0.1:8787
3) Optional: run start_console.bat and start_gui.bat
